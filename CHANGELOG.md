All notable changes to this project will be documented in this file.

## [0.1.0.1] - 2026-03-24

### Added
- Initial release
- Sandboxed execution engine with resource limits
- Unified interface for 25+ reverse engineering tools
- MCP-compatible tool schemas
- CLI interface
- Configurable tool loading via YAML
- Comprehensive documentation for all tools
- Full pytest suite with >80% coverage
- Linting with ruff and type checking with mypy
- CI/CD workflow with GitHub Actions
